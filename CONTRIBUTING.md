# Contributing to rrchecklist

## Current maintenance status

This package is offered as a starting point rather than an actively maintained project. We are not currently accepting pull requests or tracking issues with a formal review process.

## Cloning and adapting

The simplest and most effective way to "contribute" to your own research workflow is to clone this repository and tailor it to your needs:

```bash
git clone https://github.com/pitherj/rrchecklist.git
```

Under the GPL-2 license you are free to modify the templates, restructure the package, or build something new from it. If you do adapt it, we ask that you cite both this package and the guidelines document it is based on (see the README for citation details).
